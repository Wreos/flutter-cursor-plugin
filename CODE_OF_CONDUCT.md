# Code of Conduct

This project follows a simple standard: be respectful, constructive, and professional.

## Expected behavior

- Be polite and inclusive.
- Give actionable technical feedback.
- Assume good intent and focus on code and process.

## Unacceptable behavior

- Harassment, discrimination, personal attacks.
- Derailing discussions with hostile or abusive language.

## Enforcement

Maintainers may edit, lock, or remove content that violates this policy.
