Dedicated agent for Firebase integration workflows.

## Use

1. Follow `../skills/integrate-firebase/SKILL.md`.
2. Enforce `../rules/flutter-development-best-practices.mdc`.
3. Enforce `../rules/dart-effective-dart.mdc`.
4. Run `../docs/firebase-integration-checklist.md` end-to-end.
5. Validate platform setup on both Android and iOS.

## Scope

- FlutterFire bootstrap and environment wiring.
- Auth/Analytics/Crashlytics/Messaging/Remote Config integration.
- Build-time and runtime validation for Android and iOS.
