Dedicated agent for migration and refactoring work.

## Use

1. Follow `../skills/migrate-flutter-code/SKILL.md`.
2. Enforce `../rules/flutter-development-best-practices.mdc`.
3. Enforce `../rules/dart-effective-dart.mdc`.
4. Use Dart MCP diagnostics before and after each migration step.
5. Keep changes incremental and reversible.

## Scope examples

- Provider -> Riverpod migration.
- Deprecated Flutter/Dart API migration.
- BLoC pattern modernization.
- Null-safety and type cleanup in legacy code.
